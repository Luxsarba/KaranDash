# Piano MiniGame

Пианино-пазл: клавиши всегда доступны, успех срабатывает по правильной мелодии.

## Как работает

1. `PianoPanel` хранит ожидаемую последовательность нот (`melodySequence`).
2. Игрок нажимает клавиши (`PianoKey`) через интеракцию.
3. Каждое нажатие сверяется с текущим шагом мелодии.
4. Ошибка сбрасывает прогресс к началу.
5. Полное совпадение вызывает `onSuccess` и (опционально) проигрывает `successMelodyClip`.

## Скрипты

- `PianoPanel.cs` — парсинг мелодии, прогресс, success-логика (без аудио-ответственности).
- `PianoKey.cs` — отдельная клавиша: визуальная вспышка, звук, передача нажатия в панель.
- `PianoSuccessAudio.cs` — отдельный компонент для звука успеха (`Play()`), подписывается на событие `PianoPanel.Success`.

## Формат нот

Поддерживаются записи вроде:

- `C4`
- `D#4`
- `Bb4`
- `A5`

Внутри панель нормализует в sharp-формат (например `Bb4 -> A#4`).

## Ключевые настройки (`PianoPanel`)

- `melodySequence` — последовательность нот через пробел/запятую.
- `onSuccess` — событие при полном совпадении.

## Ключевые настройки (`PianoKey`)

- `noteId` — нота клавиши.
- `normalColor`, `activeColor`, `flashDuration` — визуал.
- `noteClip` или генерация тона — звук клавиши.
- `allowMouseClick` — опциональный mouse fallback (обычно выключен).

## Ключевые настройки (`PianoSuccessAudio`)

- `successClip` — клип, который проигрывается на успех.
- `successVolume` — громкость one-shot.
- `audioSource` — источник (если не указан, берётся `AudioSource` на объекте).
