# Memory MiniGame

Игра на запоминание последовательности кнопок.

## Как работает

1. Игрок запускает игру через `MemoryGameTrigger`.
2. `MemoryPanel` генерирует случайную последовательность.
3. Панель показывает часть последовательности (по раундам).
4. Игрок повторяет через интеракцию.
5. За правильный ввод длина раунда увеличивается.
6. При полной последовательности вызывается `onSuccess`.
7. При ошибке/таймауте вызывается `onFail` и (опционально) автоперезапуск.

## Скрипты

- `MemoryPanel.cs` — основная логика раундов, таймера, успеха/ошибки.
- `MemoryButton.cs` — кнопка, визуал, подсветка, нажатие.
- `MemoryGameTrigger.cs` — точка входа, запускает `StartGame()`.

## Ключевые настройки (`MemoryPanel`)

- `sequenceLength` — длина полной последовательности.
- `startRoundLength` — стартовая длина раунда.
- `buttonDisplayTime`, `pauseBetweenButtons` — темп показа.
- `inputWindowTime` — таймаут на ввод игрока.
- `autoRestartOnFail`, `restartDelay` — рестарт после ошибки.
- `onSuccess`, `onFail` — внешние действия в сцене.

## Интеграция с игроком

- Нажатия идут через `PlayerInteraction` (`E`) и `MemoryButton.TryPressFromInteraction()`.
- Пока активен Memory-раунд и открыт ввод, взаимодействие разрешается даже при некоторых общих блокировках.
